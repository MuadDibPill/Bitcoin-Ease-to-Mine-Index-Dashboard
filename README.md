# ⛏️ Ease to Mine Index Dashboard

> Dark-themed financial dashboard analyzing Bitcoin mining conditions across 19 jurisdictions

![Python](https://img.shields.io/badge/Python-3.11-3776AB?style=flat-square&logo=python&logoColor=white)
![Streamlit](https://img.shields.io/badge/Streamlit-1.32-FF4B4B?style=flat-square&logo=streamlit&logoColor=white)

## 📊 Features

| Section | Content |
|---------|---------|
| **Overview** | Global rankings, KPIs, regional heatmap |
| **Fiscal** | Tax environment, subsidies, CIT rates |
| **Permitting** | Construction permits, EIA, zoning |
| **Legal** | Regulatory framework, outlook |
| **Energy** | Grid access, connection times |
| **Customs** | Import process, VAT, tariffs |

## 🚀 Deploy

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📈 Data

Survey of 48 practitioners across 19 jurisdictions (Dec 2025 - Feb 2026)
