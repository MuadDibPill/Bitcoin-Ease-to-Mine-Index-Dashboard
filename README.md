# ⛏️ Ease to Mine Index (EMI) Dashboard

Interactive dashboard visualizing Bitcoin mining conditions across 19 jurisdictions worldwide.

## 🎯 Features

- **Overview** — Global EMI ranking and KPIs
- **Fiscal** — Tax environment, subsidies, CIT rates
- **Permitting & Licensing** — Construction permits, EIA, zoning
- **Legal** — Regulatory framework and outlook
- **Energy & Grid** — Grid access, connection times, prices
- **Customs & Tariffs** — Import procedures, VAT, tariffs

## 🚀 Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Data Source

Survey of 48 industry practitioners (Dec 2025 - Feb 2026)

Research by [Hashlabs](https://hashlabs.io)
