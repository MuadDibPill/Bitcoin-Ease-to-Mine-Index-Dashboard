# Ease to Mine Index Dashboard

Interactive dashboard analyzing Bitcoin mining conditions across 19 jurisdictions.

## Features

- Global scores with dimension filter
- EMI ranking with gradient colors  
- Score heatmap by section
- Survey respondents pie chart
- Hashrate treemap (Q1 2025 vs Q1 2026)
- Full data table with download

## Deploy

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Data

Survey of 48 practitioners (Dec 2025 - Feb 2026)

Research by [Hashlabs](https://hashlabs.io)
